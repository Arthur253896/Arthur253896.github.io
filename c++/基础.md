# 句法
## cin&&cout
C++ 中的输入与输出可以看做是一连串的数据流，输入即可视为从文件或键盘中输入程序中的一串数据流，而输出则可以视为从程序中输出一连串的数据流到显示屏或文件中。
```c++
#include <iostream>
using namespace std;
int Add(int x,int y){
    return x+y;
}
int main(){
    int x,y,z;
    cout<<"请输入x值："<<endl;
    cin>>x;
    cout<<"请输入y值："<<endl;
    cin>>y;
    z=Add(x,y);
    cout<<"x+y="<<z<<endl;
}
```

## bool类型
在cpp中bool类型以旧和c中一样，返回0和1，占一个字节；
```c++
int main(){
    bool flag;
    int a,b;
    flag=a>b;
    cout<<"flag="<<flag<<endl;
}
```
输入3，4；flag=0；
输入3，2，flag=1；
也可以定义flag为false或true；
```c++
int main(){
    bool flag = true;
    if(flag){
        cout<<"true"<<endl;
    }else{
        cout<<"false"<<endl;
    }
    flag = false;
    if(flag){
        cout<<"true"<<endl;
    }else{
        cout<<"false"<<endl;
    }
    return 0;
}
```
输出true，false；
