# 两个数组的交集 (opens new window)

题目描述：给定两个数组 nums1 和 nums2 ，返回 它们的交集 。输出结果中的每个元素一定是 唯一 的。我们可以 不考虑输出结果的顺序 。 例如{1，2，2，1} {2，3}，return{2}

给定数的范围我建立一个record数组用来记录，遍历第一个数组，出现哪个数组，相应位置++；遍历第二遍数组的时候当前位置数>0;我设置成-1，记录为相交元素，最后用一个数组记录返回。

vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        int record[1000]={0};
        for(int i=0;i<nums1.size();i++){
            record[nums1[i]]++;
        }
        for(int i=0;i<nums2.size();i++){
            if(record[nums2[i]]>0){
                record[nums2[i]]=-1;
            }
        }
        vector<int> result;
        for(int i=0;i<1000;i++){
            if(record[i]==-1){
                result.push_back(i);
            }
        }
        return result;
    }
