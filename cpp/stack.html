# Stack的一些操作

一个例子：

stack<int> myStack;//初始化一个栈
        for(int i=0;i<s.size();i++){
            myStack.push(s[i]);//push操作
        }
        for(int i=0;i<s.size();i++){
            char k=myStack.top();//赋值操作
            myStack.pop();//弹出操作
            s[i]=k;
        }
    myStack.empty()//判空操作

# 进栈数据类型

对于字符串，不仅可以进栈单个字母，可以将整个字符串都进栈，如：

stack<int> myStack;
string s="killer"
string ss="queen"
myStack.push(s)
myStack.push(ss)
