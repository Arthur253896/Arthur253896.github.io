# map

map是C++ STL（标准模板库）中的一种关联容器，用于存储键值对。map中的元素按照键的大小自动排序，并且键必须是唯一的。map容器提供了查找、插入和删除元素的高效方法，因此在实际开发中被广泛使用。

以下是一些map容器的基本用法：

# 定义和初始化

定义和初始化map容器的方式有很多种，下面是一些常见的方式：

#include <iostream>
#include <map>
#include <string>

int main() {
    // 定义一个空的map容器
    std::map<std::string, int> m1;

    // 使用初始化列表定义map容器
    std::map<std::string, int> m2 = {{"apple", 1}, {"banana", 2}, {"orange", 3}};

    // 使用迭代器定义map容器
    std::map<std::string, int> m3(m2.begin(), m2.end());

    // 使用拷贝构造函数定义map容器
    std::map<std::string, int> m4(m3);

    return 0;
}

# 插入元素

使用insert()函数可以向map容器中插入一个键值对：

std::map<std::string, int> m;
m.insert(std::make_pair("apple", 1));

也可以使用下标运算符[]来插入元素：

std::map<std::string, int> m;
m["apple"] = 1;

如果键已经存在，则插入操作会被忽略。

# 访问元素

使用下标运算符[]可以访问map容器中的元素：

std::map<std::string, int> m = {{"apple", 1}, {"banana", 2}, {"orange", 3}};
std::cout << m["apple"] << std::endl; // 输出1

如果键不存在，则会自动插入一个默认值，例如0或空字符串。

使用find()函数可以查找map容器中的元素：

std::map<std::string, int> m = {{"apple", 1}, {"banana", 2}, {"orange", 3}};
auto it = m.find("apple");
if (it != m.end()) {
    std::cout << it->second << std::endl; // 输出1
}

# 删除元素

使用erase()函数可以删除map容器中的元素：

std::map<std::string, int> m = {{"apple", 1}, {"banana", 2}, {"orange", 3}};
m.erase("apple");

也可以使用迭代器删除元素：

std::map<std::string, int> m = {{"apple", 1}, {"banana", 2}, {"orange", 3}};
auto it = m.find("apple");
if (it != m.end()) {
    m.erase(it);
}

# 遍历元素

可以使用迭代器遍历map容器中的元素：

std::map<std::string, int> m = {{"apple", 1}, {"banana", 2}, {"orange", 3}};
for (auto it = m.begin(); it != m.end(); ++it) {
    std::cout << it->first << " " << it->second << std::endl;
}

上述代码输出map容器中所有的键值对。
