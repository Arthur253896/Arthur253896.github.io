# JAVA_HOME 环境变量

先找到Java安装路径：

 $sudo update-alternatives --config java

输出：

/usr/lib/jvm/java-8-openjdk-amd64

在 /etc/enviroment文末添加：

JAVA_HOME="/usr/lib/jvm/java-8-openjdk-amd64"

修改当先shell生效：

$source /etc/environment

验证JAVE_HOME:

$echo $JAVA_HOME

然后就能看到java路径了

# 修改密码

一开始下载mysql会设置一个随机账号和密码，在/etc/mysql/debian.conf可见，进入较麻烦。 在输入密码进入mysql后依次输入：

mysql> show databases;
mysql> use mysql;
mysql> update user set authentication_string=PASSWORD("root") where user="root";

mysql> update user set plugin="mysql_native_password";
mysql> flush privileges;
mysql> quit;

其中PASSWORD后面的root可以为其他自定义的密码， 在终端中输入　sudo /etc/init.d/mysql restart;　命令，重启ｍｙｓｑｌ服务

卸载ｍｙｓｑｌ的命令：

sudo apt purge mysql-*

sudo rm -rf /etc/mysql/ /var/lib/mysql

sudo apt autoremove

安装ｍｙｓｑｌ的命令

sudo apt-get install mysql-client mysql-server

查看mysql状态

sudo service mysql status

查看mysql服务器端口

sudo netstat -tap |grep mysql

开启、停止和重启mysql服务

sudo service mysql start

sudo service mysql stop

sudo service mysql restart

安装后要打开所有人可访问权限，不然只能在root权限下才可连接数据库，导致一些可视化工具和idea连接不上

$su

进入root

#mysql
>ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY ' 你的密码';
>FLUSH PRIVILEGES;
