# 详解set容器的用法

C++中的set是STL（标准模板库）中的一个容器，用于存储一组自动排序的唯一元素。set容器的主要特点包括：

1、自动排序：set中的元素会根据其值自动进行排序，而且每个元素只会出现一次

2、快速查找：set中的元素是按照一定顺序排列的，因此可以快速查找指定元素。

3、插入和删除元素：可以在set中插入和删除元素，这些操作都不会影响set中其他元素的顺序。

# set容器的基本用法：

1、创建set对象

std::set<type> name;

2、插入元素

name.insert(elment)

3、删除元素

name.erase(element)

4、查找元素

可以使用find()成员函数查找指定元素是否存在于set容器中，该函数返回一个迭代器。如果找到了指定元素，则返回指向该元素的迭代器；如果没有找到，则返回指向set末尾位置的迭代器。以下是使用find()函数查找元素的基本方法：

std::set<int> my_set = {1, 2, 3, 4, 5};

// 查找元素
auto it = my_set.find(3);
if (it != my_set.end()) {
    std::cout << "Element found in set\n";
} else {
    std::cout << "Element not found in set\n";
}

5、遍历元素

std::set<int> my_set = {1, 2, 3, 4, 5};

// 使用迭代器遍历set容器
for (auto it = my_set.begin(); it != my_set.end(); ++it) {
    std::cout << *it << " ";
}

# 其他成员函数

除了上述基本用法外，C++ STL中的set容器还提供了许多其他常用成员函数，例如：

size()：返回set容器中元素的个数。

empty()：判断set容器是否为空。

clear()：清空set容器中所有元素。 count()：返回指定元素在set容器中出现的次数。

以上就是C++ STL中set容器的基本用法。需要注意的是，由于set容器是自动排序的，因此其插入、删除和查找操作都比较高效

# 对比vecot

1、数据结构：vector是一个动态数组，而set是一个自动排序的唯一元素集合。vector中的元素按照它们在数组中的顺序存储，而set中的元素按照它们的值自动排序。

2、元素的唯一性：vector中的元素可以重复，而set中的元素是唯一的。如果向set中插入一个已经存在的元素，则插入操作会被忽略。

3、插入和删除操作：向vector中插入或删除元素时，可能需要移动其他元素以保持数组的连续性。而向set中插入或删除元素时，不需要移动其他元素。

4、查找操作：在vector中查找元素时，需要使用线性搜索算法。而在set中查找元素时，可以使用二分查找算法，因为元素已经按照值排序。

5、迭代器：vector和set都支持迭代器，但它们的迭代器有所不同。vector的迭代器支持随机访问，因此可以像使用指针一样进行加减操作。而set的迭代器只支持双向访问，因此不能进行随机访问
