# 字符串操作

cpp中字符串被重写了,用数组来存储字符串，所以有一些操作很方便：

string s="sdadsafsadsada";
s.size()//返回字符串大小
for(int i=0;i<s.size();i++){
    s[i]='a';
}//输出s为aaaaaaaaaaaaa对应上面的长度
